| физлицо, физическое лицо, физические лица | Catalog_ФизическиеЛица |
| должность, должности | Catalog_ДолжностиОрганизаций |

## Важные особенности

- Поле «ФИО» у сотрудника: `Description` (у Catalog_Сотрудники Description содержит ФИО)
- Поле «ФИО» у физлица: `Description` (у Catalog_ФизическиеЛица)
- Сотрудник связан с физлицом через поле `ФизическоеЛицо_Key`
- Организация в документах — поле `Организация_Key`
- Подразделение в документах — поле `ПодразделениеОрганизации_Key`
- Поле «Дата приёма» у сотрудника: в регистре, не в справочнике — используй Document_ПриёмНаРаботу
